# Helper Printer Документация
## 1. Расположение файла
Для корректной работы программы `Helper Printer` требуется скопироровать папку в корневой каталог __Temp__ по данному пути:
```
C:\Temp\
```
## 2. Запуск программы
- Обязательным условием для корректной работы помошника требуется открывать ее от имени __СЕРВИСНОЙ УЧЕТНОЙ ЗАПИСИ__
![img1](./img/item1.jpg)

## 3. Использование программы
- Ввести имя компьютера в поле, предварительно скопировав его из принтера<br/>![img2](./img/item2.jpg)
- Ввести IP адрес принтера<br/> ![img3](./img/item3.jpg)
- Выбрать необходимый принтер. Высветится поле для ввода имени принтера. В некоторых принтерах программа потребуется выбрать конкретный драйвер для установки<br/>![img4](./img/item4.jpg)
- Нажать на кнопку `OK`. После этого будет запущен процесс установки принтера. Как принтер будет подключен вылезет соответствующее сообщение об успешном результате
## 4. Принцип работы Heplper Printer
### 4.1 О программе
Программа написана на скриптовом языке PowerShell. Основной функцией данной программы является выполнение сценария по установке принтера, путем установления сессии с удаленным АРМ.
